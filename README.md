# 📄 Automated Student Performance Report Generator

This is a **Python-based desktop application** that automatically generates a **PDF report** of student performance using sample marks data.
The application uses a **GUI (Tkinter)** to make report generation simple and user-friendly.



## 🎯 Project Objective

To automate the process of:

* Analyzing student marks
* Calculating average, highest, and lowest scores
* Generating a professional **PDF report** with summary and table format



## ✨ Features

* Simple GUI using Tkinter
* Automatic calculation of:

  * Average marks
  * Highest marks
  * Lowest marks
* Generates a well-formatted **PDF report**
* Displays student marks in a styled table
* One-click report generation



## 🛠️ Technologies Used

* **Python**
* **Pandas** – for data handling
* **ReportLab** – for PDF generation
* **Tkinter** – for GUI development

## ⚙️ Requirements

Ensure Python (3.x) is installed.

Install required libraries:

bash
pip install pandas reportlab

*(Tkinter comes pre-installed with most Python distributions)*



## ▶️ How to Run the Application

1. Open terminal / command prompt
2. Navigate to the project directory
3. Run the program:
bash
python student_report_generator.py



## 🖥️ How It Works

1. The GUI window opens
2. Click **“Generate Report”**
3. A PDF file named `student_report.pdf` is created
4. A success message is displayed



## 📊 Report Contents

The generated PDF includes:

* Report title
* Brief description
* Average marks
* Highest marks
* Lowest marks
* Student marks table with styling
